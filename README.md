### System Requirements
MacOSX or Linux

### Installation
- Get node + npm from https://nodejs.org/en/download/
- cd into sentimentAnalysisProblems directory
- npm install

### Run Program
- cd into sentimentAnalysisProblems directory if not already in it
- node sentimentAnalysis.js
