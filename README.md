## Sentiment Analysis and more

### System Requirements
Mac OSX or Linux

### Installation
- Get node + npm from https://nodejs.org/en/download/ and go through installation process of it.
- Do the following from your bash shell
```
cd into sentimentAnalysisProblems
npm install
```

### Running The Program
- Do the following from your bash shell
```
cd into sentimentAnalysisProblems
npm test
```
